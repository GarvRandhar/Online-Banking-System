# Online-Banking-System
A full-featured online banking system built with Flask. Supports savings &amp; current accounts, loans, credit cards, investments (FD/RD/mutual funds), bulk payments, 2FA security, automated EMI processing, and email notifications. Includes dashboard analytics, transaction history, and debit card management.
